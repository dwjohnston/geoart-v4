# geoplanets.io 

An interactive geometric art generator. 

Live site: https://geoplanets.io

## Features

- Several interactive geometric generators
- Users can download their image, or share their image to social media. 
- When users access the share page - the image is automatically shared on the [geoplanets.io twitter feed.](https://twitter.com/geoplanetsio)

## Technical Architecture 

- Built with ReactJS
- Hosted on Google Firebase
- Uses [blacksheep-react-canvas](https://www.npmjs.com/package/blacksheep-react-canvas) for canvas rendering.
